# Desafío 2 - Condiciones y Operadores Lógicos 🚀

Este repositorio contiene la resolución de los 3 problemas independientes planteados para evaluar el uso de estructuras de control condicionales (`if`, `else if`, `else`), operadores lógicos y manipulación del DOM mediante JavaScript.

## 🛠️ Tecnologías y Conceptos Implementados

* **Manipulación Dinámica de Estilos:** Uso de condicionales en JavaScript para evaluar estados de CSS en tiempo real (añadir o quitar bordes mediante clicks).
* **Validación Matemática Condicional:** Extracción y conversión de datos de formularios (`parseInt`) para realizar operaciones lógicas que restringen límites cuantitativos (máximo de 10 elementos).
* **Control de Flujo Complejo:** Uso de estructuras `if / else if / else` acopladas para la verificación de patrones de strings específicos (claves de acceso).
* **Estructura Organizada:** Separación modular de archivos estáticos y de lógica de programación dentro del directorio corporativo `/assets`.
